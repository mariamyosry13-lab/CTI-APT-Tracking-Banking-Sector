# APT Tracking and Profiling for Banking Sector in MENA

This project aims to track and profile Advanced Persistent Threat (APT) groups targeting the banking and financial sectors in Egypt and the MENA region.

## Tools Used
- OpenCTI
- MISP
- AlienVault OTX

## Objectives
- Identify APT groups targeting banks
- Analyze TTPs using MITRE ATT&CK
- Collect Indicators of Compromise (IOCs)
- Integrate threat intelligence platforms
