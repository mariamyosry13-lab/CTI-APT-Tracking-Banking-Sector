# CTI APT Tracking for Banking Sector in MENA

## Project Overview
This project focuses on Cyber Threat Intelligence (CTI) analysis to track and profile Advanced Persistent Threat (APT) groups targeting the banking sector in the Middle East and North Africa (MENA) region.

The project aims to analyze threat actors, attack techniques, and indicators of compromise (IOCs) related to cyberattacks against financial institutions.

## Objectives
- Analyze cyber threats targeting banks in the MENA region
- Identify Advanced Persistent Threat (APT) groups involved in financial cybercrime
- Map attacker behavior using the MITRE ATT&CK framework
- Collect and analyze Indicators of Compromise (IOCs)
- Investigate data leaks and threats from the dark web
- Provide detection and mitigation strategies for banking systems

## Project Scope
The project focuses on cyber threats affecting financial institutions such as banks, payment systems, and financial services organizations across the MENA region.

## Tools and Frameworks Used
- MITRE ATT&CK Framework
- Cyber Threat Intelligence (CTI) Reports
- CrowdStrike Threat Intelligence
- Open-source threat intelligence platforms
- Dark web monitoring techniques

## Project Structure

### 1. Group IP Report
Analysis of suspicious and malicious IP addresses associated with attacks targeting the banking sector.

### 2. MITRE ATT&CK Analysis
Mapping attacker techniques and tactics using the MITRE ATT&CK framework to understand how cyber attacks are executed.

### 3. CTI Reports
Collection and analysis of Cyber Threat Intelligence reports related to APT groups and financial sector attacks.

### 4. Banking Sector – MENA Region
Study of cyber threats targeting banks and financial organizations in the MENA region.

### 5. CrowdStrike Intelligence
Review and analysis of threat intelligence reports published by CrowdStrike regarding cybercriminal groups targeting financial institutions.

### 6. Dark Web Scrubbing
Investigation of potential data leaks, stolen financial information, or compromised credentials available on dark web marketplaces.

### 7. Tracking Banking Sector Threats
Monitoring ongoing cyber threats and campaigns targeting the financial sector.

### 8. Detection and Mitigation
Providing cybersecurity strategies and defensive techniques to detect and prevent attacks on banking systems.

## Expected Outcomes
- Identification of major cyber threats targeting banks
- Analysis of attacker techniques using MITRE ATT&CK
- Documentation of threat intelligence findings
- Security recommendations to improve banking sector defenses

## Team
